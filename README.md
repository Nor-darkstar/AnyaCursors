# AnyaCursors

### Installation

AnyaCursors is installed by running the following commands in your terminal.

| Method    | Command                                                                                           |
| :-------- | :------------------------------------------------------------------------------------------------ |
| **curl**  | `sh -c "$(curl -fsSL )https://raw.githubusercontent.com/Nor-darkstar/AnyaCursors/refs/heads/master/Tools/Oninstaller.sh"` |
